# NutriMind 🧠

NutriMind is a personal AI-powered productivity system designed to manage learning, data, and scheduling in one interface.

## Features
- Mind Engine (AI assistant powered by local LLaMA)
- Data Manager (structured knowledge storage)
- Calendar system with event tracking
- Quick Search launcher (Google integration)

## Tech Used
- HTML
- CSS (custom UI system)
- JavaScript (vanilla)
- Local LLaMA integration

## Purpose
Built as a personal operating system to support continuous learning, focus, and productivity.

## Status
Stable version – optimized for desktop use.
